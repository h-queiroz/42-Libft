# My Libft
This is the first real project from the Core Common curriculum from 42SP where we are supposed to create our own versions of some of the most common functions from the C Library.
These functions are:
- isalpha
- isdigit
- isalnum
- isascii
- isprint

# How to compile and test
To compile all functions you simply have to on the root of the project and run the command `make`, which should compile the most basic functions into a single static library, which is gonna be used for the most complex functions.
After that, you can enter the ***tests*** directory and run `make` again which is gonna run the test suite with a couple of examples that I created personally to test each function. You can add your own tests by editing the file **tests_libft.c** and running `make` again.
